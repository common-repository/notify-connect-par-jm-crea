== Description ==

Notify connect créé par JM Créa permet d’être notifié par email et sms (pour les abonnés freemobile) lorsqu’un administrateur, éditeur, auteur, contributeur ou shopmanager se connecte sur le backoffice.

Le plugin est parametrable.

N’hésitez pas à laisser vos commentaires et à noter ce plugin.

== Installation ==

1. Uploadez `notify—connect-par-jm-crea` dans le dossier `/wp-content/plugins/` de votre site
1. Activez ’Notify connect’ depuis le menu « Extensions » de Wordpress
1. Rendez-vous ensuite sur ‘Notify connect’ dans votre menu à gauche.
